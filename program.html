Träningsprogram

Skräddarsydda träningsprogram designade just för dig. Varje program är unikt anpassat efter dina mål, nuvarande fitnessnivå och preferenser. Vi använder vetenskapliga metoder för maximal effektivitet.

Muskeluppbyggnad

Program fokuserat på att bygga lean muscle mass. Perfekt träning kombinerat med rätt näring för optimal resultat.

Fettnedmontering

Väl designat program för att tappa fett medan du behåller din muskelmassa. Snabba resultat utan att hungra.

Konditionsökning

Träna din kardiovaskulära hälsa och uthållighet. Perfekt för löpare och idrottare som vill förbättra sin form.

Styrketräning

Fokuserat på att öka din maximal styrka. Baserat på proven träningsmetoder för kraftfolket.

Funktionell träning

Träning som förbättrar din dagliga funktion och rörlighet. Perfekt för alla åldrar och fitnessnivåer.

Rehab & Återhämtning

Skonsam träning för att återhämta sig från skador. Personlig vägledning för säker återgång till träning.
